# Microwave Using Tkinter Microwave

An educational Python interface meant to mimic a real microwave. Demonstrates basic Python knowledge by:

- Creating and displaying a Tkinter interface with interactable buttons.
- Thoughtfully positioning elements on the application into a user friendly grid.
- Displaying an image and changing it based on conditionals.
- Using event loop scheduling functions to create a functioning timer.
- String manipulation and casting to keep track of minutes and seconds.
- Using winsound to play tones at specific frequencies for a set duration.
